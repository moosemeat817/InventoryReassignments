# InventoryReassignments
This is a mod for The Long Dark by Hinterland Studio, Inc.

## Description
Reassign various gear items to different inventory categories.

## Automatic Reassignments
Moved to Materials - ArrowHead, ArrowShaft, BrokenArrow, BrokenArrowHardened, CarBattery, Fuse, GoldNugget, Hook, Line, RevolverAmmoCasing, RifleAmmoCasing, SpearHead, WireBundle
